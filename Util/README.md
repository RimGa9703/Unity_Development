# Util
    개발 편의성을 위한 Unil 모음 폴더

## AssetManager.cs
    ResourceLoad로 부른 에셋들의 재활용을 위한 스크립트

## AssetPath.cs
    리소스들이 저장된 경로를 담은 스크립트

## MonoSington.cs
    디자인 패턴 Singletone이 필요하며 Monobehavior를 상속받아야하는 경우인 Singletone 스크립트
    
## Singletone.cs
    디자인 패턴 Singletone을 쉽게 구현하기 위한 스크립트

## UnityPoolManager.cs
    유니티 2021 버전부터 지원하는 ObjectPool을 응용하여 만든 풀매니저 스크립트

## FollowCamera.cs
    타겟이 된 대상을 따라가는 카메라 스크립트

## SerializableDictionary.cs
     Unity 직렬화 시스템에서 Dictionary는 기본적으로 지원되지 않기 때문에,
     직렬화 가능한 형태(List 기반)로 Dictionary 기능을 제공하기 위한 스크립트
     
## StringMaker.cs
    문자열을 빈번하게 조합해야 하는 상황(UI 텍스트, 로그, 디버그 문자열 등)에서
    StringBuilder를 재사용하기 위한 스크립트
