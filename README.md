# Unity_Development

이 저장소는 Unity 게임 개발 중 자주 사용하는 기능, 편의성 향상 스크립트, 혹은 게임에 필요한 핵심 기능 모아두기 위한 공간입니다.

주로 C#(.cs) 파일로 구성되어 있으며, 각 기능은 폴더별로 정리되어 있습니다.

더불어 코드 스타일 체킹용 레포지토리이기도 합니다. :)

## Repository Structure

- `FSM` - FSM 관련 스크립트 모음
- `Projectile` - 발사체 관련 스크립트 모음
- `UI` - UI 관련 스크립트
- `Util` - 개발 편의성 도우미 유틸 스크립트
- `TileMapBuildingSysyem` - 2D타일맵 기반 건설 경영 시뮬레이션 게임을 위한 시스템 모음(Task 시스템, 타이쿤 시스템 등)
